# Hallo-dott
###Input
```
1

34*

0001 **
0011
0110
```

```
* 3 - line
* 4 - column

** 1 - white pixel
** 0 - black pixel
```


###Output
```
3 2 1 0
2 1 0 0
1 0 0 1
```

## Installation
In the project directory, you can run:

1. Clone project
```
git clone https://github.com/Almazatun/hallo-dott.git
```
2. Download dependencies
```
npm install
```
3. Start dev server
```
npm run dev
```
4. Tests
```
npm run test
```
